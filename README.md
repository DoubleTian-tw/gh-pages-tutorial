# What is gh-pages ?
- 是github提供的免費靜態網頁使用套件
- 透過此套件能夠建立數個不同的專案，只需要切換不同名稱即可，每一個repo可以是一個gh-pages
- 路徑顯示方式 : https://[你的帳號名稱].github.io/[專案名稱]
